# Calculator
A calculator that allows users to do basic operations. This application is a TDD with jest and enzyme. 

## How To Start 
```
clone the repo
npm install 
npm start
npm run build
```
In order to run tests 
```
npm run test
```
or
```
npm run test:c
```


## Tech Used
React
Express
Jest
Enzyme
TDD (Test Driven Development)
Scss
